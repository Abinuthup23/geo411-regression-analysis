# geo411-Regression Analysis

Geography 411 | Homework 4

## Overview
This repository contains the R code and analysis for Geography 411 Homework 4. The purpose of this assignment was to use regression analysis to examine the relationship between SAT score and GPA, and the relationship between mosquito development sites and Aedes aegypti pupae.

## Methods
- Scatterplots 
  Used to visualize the relationship between the variables in each dataset.

- Linear regression 
  Used to estimate the relationship between SAT and GPA, and between development sites and pupae.

- Diagnostic plots 
  Used to check model assumptions and identify possible outliers.

- Log transformation 
  Used on the mosquito data to improve model fit and reduce skewness.

- Log-log regression 
  Used to examine the percent change relationship between development sites and pupae.

## Key Findings
- SAT score was a statistically significant predictor of GPA.
- The SAT model explained a large amount of variation in GPA.
- Development sites were a significant predictor of mosquito pupae.
- Removing the outlier changed the slope and improved the model fit.
- The log-log mosquito model had the strongest fit out of the mosquito models.

## Tools
- RStudio 
  Used to run the regression models, create plots, and analyze the results.

- GitHub 
  Used to store and organize the project files.

## Reflection
This assignment helped me better understand how regression works and how to interpret slope, R-squared, and significance tests. It also showed how outliers and transformations can affect model fit and make a model more useful.
